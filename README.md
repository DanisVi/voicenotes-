# 🔐 VoiceNotes Pro

Защищенное приложение для голосовых заметок с Face ID/Touch ID и iCloud синхронизацией.

## ✨ Возможности

- 🎤 **Голосовой ввод** — говорите команды, не печатайте
- 🔒 **Face ID / Touch ID** — биометрическая защита
- 🔢 **PIN-код** — 4-значный пароль с возможностью смены
- ☁️ **iCloud Sync** — резервное копирование в JSON
- 📁 **Группы и главы** — умная сортировка записей
- 🔄 **Автоблокировка** — при сворачивании приложения
- 💾 **Автосохранение** — данные не теряются

## 🚀 Установка на iPhone

1. Откройте ссылку в **Safari**
2. Нажмите **"Поделиться"** → **"На экран «Домой»"**
3. Готово! Приложение работает как нативное

## 🎤 Голосовые команды

> "Группа Работа, Глава Встречи, обсудить дизайн с командой"

Приложение автоматически создаст группу, главу и запись.

## 🔒 Безопасность

- **Шифрование**: данные хранятся локально в защищенном хранилище
- **Биометрия**: Face ID / Touch ID (можно отключить)
- **PIN-код**: 4-значный пароль с возможностью смены
- **Автоблокировка**: приложение блокируется при сворачивании

## ⚙️ Настройки

Вкладка **"Защита"** позволяет:
- Изменить PIN-код
- Включить/отключить Face ID
- Настроить автоблокировку
- Управлять iCloud синхронизацией
- Заблокировать приложение вручную

## 📱 Системные требования

- iOS 14+
- Safari (для установки на домашний экран)
- Микрофон (для голосового ввода)
