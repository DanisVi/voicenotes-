# VoiceNotes Pro 🔐

Защищенное приложение для голосовых заметок с Face ID/Touch ID.

## Возможности

- 🎤 **Голосовой ввод** — говорите, не печатайте
- 🔒 **Face ID / Touch ID / Пароль** — биометрическая защита
- ☁️ **iCloud Sync** — синхронизация между устройствами
- 📁 **Группы и главы** — умная сортировка
- 💾 **Автосохранение** — ничего не потеряется

## Установка на iPhone

1. Откройте ссылку в Safari
2. Нажмите "Поделиться" → "На экран «Домой»"
3. Готово! Работает как нативное приложение

## Команды голосового ввода

> "Группа Работа, Глава Встречи, обсудить дизайн с командой"

Приложение автоматически распределит по группам и главам.

---

**Безопасность:** Данные шифруются и хранятся локально. Для синхронизации используется iCloud.
